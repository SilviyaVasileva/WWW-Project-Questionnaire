Questionnaire

Project for WWW-14 

The first page shown is index.php.
If the user doesn't have a registration then he/she need to register.
After login the user can solve tests and questionnaries.

Steps of installation:
1. Open XAMPP
2. Load Apache and MySQL
3. Open phpMyAdmin and create a database named quiz1
4. Import the database named quiz1 from folder database
5. Move the project folder in xampp/htdocs
6. Open localhost
7. Open the project folder
